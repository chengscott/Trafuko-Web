### Software Studio
# Trafuko | 垃圾話

## Prerequisites

* node
* yarn

## Environment Setup

```bash
yarn # install dependencies
```

## Run in development

```bash
yarn run dev
```

## Run in production

```bash
NODE_ENV=production
yarn run build
yarn start
```

## Test

```bash
yarn run test
```

## Lint

```bash
yarn run lint
```
